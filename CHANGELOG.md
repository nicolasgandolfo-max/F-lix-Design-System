# Changelog

All notable changes to @felix/ui will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-20

### Added

- Initial release of @felix/ui
- Button component with 9 variants (default, primary, secondary, tertiary, destructive, outline, ghost, link, link-secondary)
- Button sizes (sm, default, lg, cta, icon variants)
- Felix Tailwind preset with full brand theme
- Dark mode support
- Storybook documentation
- TypeScript definitions
- ESM and CJS module formats
- `cn` utility function for class merging

### Features

- Complete Felix brand color palette
- Custom Felix fonts (Plain-500, Saans family)
- Responsive design tokens
- Accessible components built on Radix UI
- Tree-shakeable exports
